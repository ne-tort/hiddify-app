# Решение по альтернативным сжатым FIB-backend’ам

**Устарело.** Актуальное состояние: один wg-like trie (`allowedips_peer.go`); см. [README.md](README.md).

Исторически рассматривались LC-trie / stride_v4 и feature-flag’и. Решение было склоняться к осторожности из-за сложности и регрессий. После перехода на перенос `allowedips` из `wireguard-go` отдельное «сжатое» ядро не используется; будущие изменения — только в рамках той же семантики LPM и тестов.
