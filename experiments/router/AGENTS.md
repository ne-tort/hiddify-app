# AGENTS — зона `experiments/router`

## Источник истины по архитектуре

**[`docs/router-architecture.md`](docs/router-architecture.md)** — целевая модель: **звезда**, статические маршруты как **пиры WG**, маршрутизация по **dst/src** и привязке сессии, **без обязательного runtime control plane**.

## Код sing-box в этой ветке

Форк: `experiments/router/hiddify-sing-box`. Текущий **l3router** в репозитории — реализация **варианта с control plane** (owner, runtime API, усложнённый стенд). Это **не** финальная архитектура под цель «как WG»; см. раздел «Отношение к текущему коду» в `docs/router-architecture.md`.
