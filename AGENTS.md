# Правила для ассистентов (репозиторий `hiddify-app`)

## Текущий фокус работ

Работаем над тремя основными направлениями:

- `hiddify-core` (форк `sing-box`): расширение и модификация ядра;
- клиент `hiddify`: GUI-оболочка над этим ядром;
- панель `s-ui`: серверная часть для клиента Hiddify с тем же ядром `sing-box` (включая переопределение образа ядра).

## Общие принципы

- Изменения должны быть точечными, с сохранением обратной совместимости.
- Приоритет: корректность, предсказуемость поведения и отсутствие регрессий.
- Не добавлять лишние фичи вне текущих задач.
- Перед завершением работ проверять сборку/тесты в затронутых частях проекта.
